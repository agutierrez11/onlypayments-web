# Reglas del Proyecto OnlyPayments

## 🚀 Regla Crítica de Despliegue (CI/CD)
**TODO EL FLUJO DE DESPLIEGUE ES VÍA CLOUD (EJ. GOOGLE CLOUD RUN / REPLIT CLOUD).** 
Todo el flujo de trabajo y despliegue de este proyecto está atado a GitHub para CI/CD hacia Cloud. 

Si el usuario (Antonio) pide "subir cambios", "desplegar", o "actualizar el proyecto", DEBES:
1. Hacer `git add .`
2. Hacer `git commit -m "mensaje"`
3. Hacer `git push origin main`

El despliegue en la plataforma Cloud se activa automáticamente a través de la integración nativa con GitHub. No uses comandos locales de despliegue en la terminal bajo ninguna circunstancia.
